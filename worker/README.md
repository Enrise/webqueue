# WebQueue
A message queue worker that forwards messages to a HTTP-interface.
Particularly useful for job queue type of systems.

## Installation

To be documented

## Usage

```bash
php bin/webqueue.php -q foo -h 192.168.59.103 -d amqp -e http://localhost:1337/
```
